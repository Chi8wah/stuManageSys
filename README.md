# stumanagesys

### 环境搭建
```
npm install
```

### 前端运行方法
在```stumanagesys```根目录使用命令行运行
```
npm run serve
```

根据命令行提示地址打开网页

### 后台运行方法
在```stumanagesys\serve```目录使用命令行运行
```
node app.js
```
